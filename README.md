# MarketPlace refund Dapp

This Daap will process refunds for e-commerce and transportation tickets.
An escrow contract will use the following rules:

## E-Commerce

* Timely manner delivery for the product

* Customer satisfaction
## Transportation tickets
* Refund will be based on no show, or late show.
* No show(no refund at all)
* Late show(refund 25%)
